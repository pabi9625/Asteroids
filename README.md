# Asteroids

A classic Asteroids arcade game built with Python and Pygame.

## About

This project was developed as part of the
[Build Asteroids using Python and Pygame](https://www.boot.dev/courses/build-asteroids-python)
course on [Boot.dev](https://www.boot.dev).

It was a hands-on exercise to reinforce core Python concepts including
object-oriented programming, game loops, collision detection, and working
with an external library (Pygame).

## Certificate

[View my completion certificate](https://www.boot.dev/certificates/e743a2fb-9b17-466a-826e-f0e2ff1b2603)

## Gameplay

- Pilot a spaceship and shoot down incoming asteroids
- Asteroids split into smaller pieces when hit
- Avoid collisions or it's game over

## Tech Stack

- Python 3
- Pygame

## Getting Started

### Prerequisites

- Python 3 installed
- Pygame installed: `pip install pygame`
